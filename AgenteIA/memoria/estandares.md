# Memoria: Estándares de Código

## HTML
- Uso de HTML5 semántico (`<header>`, `<main>`, `<section>`, `<footer>`)
- Todos los interactivos deben tener IDs descriptivos
- Imágenes con atributo `alt` obligatorio para accesibilidad

## CSS (Vanilla)
- Diseño responsivo (Mobile First)
- Variables CSS para la paleta de colores
- Uso de Flexbox y Grid para layouts
- Evitar frameworks pesados si no es estrictamente necesario

## JavaScript
- Vanilla JS preferido para micro-interacciones
- Gestión de estado mínima para el carrito de compras
