# Release v1.12.6 - Refinamiento UI y Reestructuración de Skills

**Fecha:** 2026-03-14
**Versión:** 1.12.6

## Cambios Realizados

### 1. Reestructuración de Documentación (Skills)
- Se renombró el skill `compilador.md` a `compila.md` para mayor brevedad.
- Se renombró el skill `frontend.md` a `diseño.md` para reflejar mejor su contenido de identidad visual y diseño.
- Se actualizaron los títulos internos y se corrigieron errores de formato (lint) en ambos archivos.

### 2. Nuevo Patrón de Diseño: "Botones de colores"
- Se agregó la definición oficial del estilo `.btn-color` en `diseño.md`.
- Implementación de estilos globales en `src/assets/main.css`.
- Este diseño utiliza fondos oscuros con bordes laterales vibrantes para indicadores de estado premium.

### 3. Mejoras en "Gestión de Equipos"
- Se aplicó el nuevo diseño de "Botones de colores" a las columnas de **Estado** y **Asistencia** en la tabla de alumnos.
- Estados ahora usan `.btn-color-info`.
- Asistencia utiliza `.btn-color-success` (Presente), `.btn-color-danger` (Ausente) y `.btn-color-tertiary` (Retardo).

### 4. Corrección en "Seguimiento PDA"
- Se optimizó la lógica del botón **Guardar Cambios**.
- Se implementó un flag `isInitializing` para evitar que el botón se habilite automáticamente al cargar los datos del documento.
- El botón ahora solo se activa cuando se detectan cambios de entrada reales por parte del usuario.
