# Skill: Web Design

Este skill define las reglas de diseño y estructura para el portal web de Arkhe Aula.

## Estándares de Código

### HTML
- Uso de HTML5 semántico (`<header>`, `<main>`, `<section>`, `<footer>`).
- Todos los interactivos deben tener IDs descriptivos.
- Imágenes con atributo `alt` obligatorio para accesibilidad.

### CSS (Vanilla)
- Diseño responsivo (Mobile First).
- Variables CSS para la paleta de colores.
- Uso de Flexbox y Grid para layouts.
- Evitar frameworks pesados si no es estrictamente necesario.

### JavaScript
- Vanilla JS preferido para micro-interacciones.
- Gestión de estado mínima para el carrito de compras.

## Identidad Visual (Website)

| Elemento | Regla |
| :--- | :--- |
| **Color Primario** | #020617 (Dark Blue) |
| **Color Acento** | #0ea5e9 (Sky Blue) |
| **Tipografía** | Inter, sans-serif |
| **Estilo** | Moderno, Limpio, Premium |

## Componentes Reutilizables

1. **Navbar**: Transparente al inicio, color sólido al hacer scroll.
2. **Hero Section**: Captura de pantalla de la app + Botón de descarga.
3. **Price Cards**: Tabla de precios con comparativa de planes.
4. **Footer**: Enlaces legales, redes sociales y contacto.
