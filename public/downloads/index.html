📋 Requisitos del Sistema

  • Windows: Windows 10 o superior (64-bit)
  • macOS: macOS 10.12 (Sierra) o superior
  • RAM: 4 GB mínimo (8 GB recomendado)
  • Almacenamiento: 500 MB libres
  • Internet: No requerido para uso básico

🔄 Actualizaciones Automáticas

Arkhe Aula verifica automáticamente si hay nuevas versiones al iniciar la aplicación. Cuando hay una actualización disponible, se te notificará y podrás descargarla e instalarla con un solo clic. Las actualizaciones se descargan directamente desde GitHub Releases.
